# Overview

**wp-module-migration** initiates the migration process for Newfold WordPress sites. It registers with the Newfold Module Loader and uses instawp/connect-helpers and wp-module-loader. Used by onboarding and other flows. Maintained by Newfold Labs. Distributed via Newfold Satis.
