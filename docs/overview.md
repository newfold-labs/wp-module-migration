# Overview

**wp-module-migration** initiates the migration process for Newfold WordPress sites. It registers with the Newfold Module Loader, uses vendored InstaWP migration utilities, and depends on wp-module-loader. Used by onboarding and other flows. Maintained by Newfold Labs. Distributed via Newfold Satis.
